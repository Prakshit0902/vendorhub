# VendorHub

VendorHub is a modern, responsive web application dashboard designed for managing vendor operations, daily needs tracking, and order analytics.

> **Note:** This project was developed strictly for **frontend demonstration purposes only**. It showcases modern UI/UX design patterns, state management, and component architecture, but does not include a functional backend or real database.

## Features

- **Interactive Dashboard:** Quick overview of recent orders, key statistics, and upcoming deliveries.
- **Theme & Localization:** Built-in contexts for dark/light mode switching and internationalization (i18n) support.
- **Order Management:** Dedicated interfaces for reviewing order summaries, adding extra items, processing returns, and pausing subscriptions.
- **Analytics View:** Clean and accessible pages for tracking metrics and performance.
- **Responsive Navigation:** Desktop sidebar and mobile bottom navigation ensuring a seamless experience across all devices.

## Tech Stack

- **Framework:** React
- **Build Tool:** Vite
- **Styling:** Tailwind CSS

## Getting Started

To run this demonstration locally:

1. Clone the repository
2. Install dependencies with `npm install`
3. Start the development server with `npm run dev`
